# Getting Started

## Available Scripts

In the project directory, you can run:

### `pnpm dev`

To start the app in dev mode.\
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### `pnpm start`

For production mode

### `pnpm test`

Run the test cases.
