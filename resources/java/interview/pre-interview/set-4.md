1. What is the output of the following program

```
int result = 0;
for(int i = 0; i < 20; ++i) {
  if(i % 5 == 0 || i % 7 == 0)
    result += i;
}
System.out.println(result);
```

a) 31
b) 45
c) 51
d) 55

Answer:c
